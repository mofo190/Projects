Use Case #1 - Menu Directory (Completed)
- Scenario:
  - User will need an interface to access the functions included within the program
  - The user will be given an initial three options (prime_menu()) which gives them the following options -> 1. View Data, 2. Comparison Calculator, 3. View Graphical Data, 4. Exit each leading to a set of sub-options, bar the fourth option Exit which will stop the program
  - Picking one option will lead to a set of sub-options each with an option to return to the previous option (return to main menu)
  - Choosing any of these sub-options (except the 'return to main menu) will call upon the specified function to complete the task
 
Improvements:

- More efficient coding method to display menus in 'main'
